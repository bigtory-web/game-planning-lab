# Work Handover Prompt

## 기본 인계 문장

```text
`game-planning-lab` 저장소를 기준으로 작업해줘.
먼저 `projects/` 안의 프로젝트 구조와 각 프로젝트의 `docs / prototype / prompts` 연결 상태를 파악해줘.
특히 현재는 `mercenary-reselect`와 `spin-miner` 프로젝트가 핵심이니, 이 두 프로젝트를 우선 확인해줘.
```

## 구조 파악용

```text
`game-planning-lab` 저장소를 기준으로 현재 작업 구조를 먼저 파악해줘.
`projects`, `prompts/common`, `templates`, `tasks`, `archive`의 역할을 기준으로 보고,
특히 `docs / prototype / prompts`가 프로젝트별로 어떻게 연결돼 있는지 확인해줘.
```

## 용병 재선택 전용

```text
`game-planning-lab/projects/mercenary-reselect`를 기준으로 봐줘.
문서, 프로토타입, 프롬프트가 같은 시스템 의도 아래 일관되게 정리돼 있는지 먼저 확인해줘.
```

## 스핀마이너 전용

```text
`game-planning-lab/projects/spin-miner`를 기준으로 봐줘.
기존 스핀마이너 관련 문서와 프로토타입이 새 저장소 구조 안에서 적절히 분리돼 있는지 확인해줘.
```

## 사용 메모

- 새 세션 시작 시 기본 인계 문장을 먼저 쓰면 됨
- 구조 점검이 필요하면 `구조 파악용` 사용
- 특정 프로젝트만 볼 때는 전용 문장 사용
